## 博客搭好了 🎉

用 **Hugo** 静态站点生成器 + **PaperMod** 主题 + **GitHub Pages** 免费托管，零成本拥有了一个个人博客。

### 步骤回顾

1. 在 GitHub 上创建 `yingkuanwang.github.io` 仓库
2. 本地用 Hugo 生成站点
3. 选 PaperMod 主题
4. 配置域名和主题参数
5. 推送代码，GitHub Actions 自动部署

以后就在这里写东西了。
